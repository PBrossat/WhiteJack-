void JeuSolo::initialisationMise ( const char  touche)

Procédure permettant de saisir la mise de départ.

Paramètres
[in]touche: la touche saisie par l'utilisateur
Renvoie
void

Exemple d'utilisation : unJeu.initialisationMise('a');

Avertissement
La mise saisie doit être inférieur au budget du joueur (2000)
Le parametre touche doit etre obligatoirement 'a' pour 1, 'z' pour 10, 'e' pour 100, 'r' pour 250 ou 't' pour 500
